	Construção do Banco Tera Comércio de Produtos S.A/q
	-> Tipagem de tabela de Bancos de Dados
		-> Char e Varchar => ex: Arvore, João, Matemática.
			--> o tipo varchar varia de acordo como numero de caracteres, enquanto char não varia e cada espaço vazio tem
				um tamnho de 1 byte
			--> Utilizar char quando onumero de caracteres da coluna nunca for variante 
				e com isso o tipo não gasta tempo variando de tamanho
		-> Float e Int => ex: 2,908; 8; 10; 3,14
		-> Blob => Arquivos como fotos e doc
		-> Text => Texto longos
		-> um banco tipado corretamente, é um banco mais leve e mais rapido
		-> O tipo ENUM existe apenas no MySQL. Em outros bancos temos que criar Constraints.
		-> limitações dos tipos 
			--> Os nulos e os inteiros 
		-> PARTES DA QUERY
			--> PROJEÇÕES
				---> será digitado SELECT
			--> Seleção 
				---> Será digitado where 
					-- EX: Traga todos os registros de RJ
